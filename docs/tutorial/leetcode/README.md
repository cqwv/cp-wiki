# 关于Leetcode

原本是一个求职刷题的网站，国服叫[力扣](https://leetcode-cn.com)。近几年来，长期举办周赛（固定在北京时间每周日的上午十点半开始），最近又增加了双周赛（每两周一次，北京时间周六的晚上十点半开始）。

赛题质量波动较大，总体难度低于其他平台。比较适合新手。
